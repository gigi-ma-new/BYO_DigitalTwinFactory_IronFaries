# Claims Processing Standard Operating Procedure
## Zurich Insurance Hong Kong — Motor Claims Division
### Version 2.4 | Last Updated: March 2024

---

## 1. Scope
This SOP covers end-to-end motor claims processing from First Notification of Loss (FNOL) to settlement payment.

## 2. Process Steps

### Step 1: Receive FNOL
- Check dedicated Claims inbox every 30 minutes
- Log claim in Excel tracker (Sheet B, Column A-F)
- Assign case reference number manually: ZUR-HK-YYYY-####

### Step 2: Initial Triage
- Open Core Policy System (CPS)
- Verify policy is active: Menu → Policy Search → Enter Policy #
- Check coverage type matches claim type
- If coverage mismatch → Refer to Senior Handler

### Step 3: Document Collection
- Send standard template email (Template #CM-03) requesting:
  - Police report
  - Repair estimate (minimum 2 quotes)
  - Photos of damage
- Log document request date in Excel tracker

### Step 4: Manual Excel Logging ⚠️
- Copy case reference to Excel Sheet B for audit trail
- Fill columns: Date Received, Handler Name, Policy #, Claim Amount, Status
- Save file to shared drive: Z:\Claims\2024\Motor\Tracker.xlsx
- **Note: This step is mandatory for quarterly audit compliance**

### Step 5: Assessment
- If claim amount < HKD 50,000 → Auto-approve pathway
- If claim amount HKD 50,000 - 200,000 → Senior review required
- If claim amount > HKD 200,000 → Refer to Claims Manager + UW
- Document decision rationale in CPS Notes field

### Step 6: Adjuster Assignment (if needed)
- For claims > HKD 100,000, assign external adjuster
- Email adjuster via Template #CM-07
- Log adjuster assignment in Excel tracker
- Follow up if no response within 3 business days

### Step 7: Settlement Calculation
- Open Calculator spreadsheet: Z:\Claims\Tools\Settlement_Calc.xlsx
- Input: Repair cost, Excess amount, Depreciation factor
- Output: Net settlement amount
- Cross-check against policy limit in CPS

### Step 8: Approval & Payment
- Generate payment instruction in CPS
- Print approval form (Form CP-12)
- Obtain physical signature from Claims Manager
- Scan signed form and attach to CPS case file
- Submit payment instruction to Finance team via email

### Step 9: Close Case
- Update status in CPS: "Settled"
- Update Excel tracker: Final amount, Close date
- File physical documents in cabinet (Room 3, Row C)
- Send closure notification to policyholder (Template #CM-11)
