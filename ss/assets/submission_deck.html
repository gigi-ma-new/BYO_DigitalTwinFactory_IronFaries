BYO Use Case
Digital Twin Factory — An Agent That Builds Agents
Main learning from experience
1. Unstructured input works — voice notes + random docs → structured twin (no templates needed). 2. Showing AI reasoning builds trust faster than hiding it (transparency > magic). 3. One non-developer built 15+ production AI skills in 2 months — citizen developers CAN build production AI.
Hyperchallenge 2026

Description and quantification of the problem

  • 80% of day spent on repetitive tasks across all departments
  • 30–50% of SOPs outdated — official process ≠ actual workflow
  • Knowledge loss when staff leave — no capture mechanism
  • 95% of staff lack technical skills for AI adoption
  • Cost: $31K/year wasted per person on manual grunt work
  • 1,000 staff HK × $31K = $31M annual opportunity

How your approach is solving the problem

  • 11 Factory Agents analyze your role via voice/docs/video → build personal AI twin in 10 minutes, zero coding
  • Twin operates daily with 4 tailored skills (Claims, Email, Reports, EXCO Prep)
  • Human approval via MS Teams — nothing sent/escalated without explicit OK
  • Self-improving: every edit permanently trains the twin (edit rate 40% → 12%)
  • Level 5 agentic: reason, plan, decide, self-correct with full transparency
  • Process Streamlining: factory finds 30–50% redundant SOP steps before building

What the results of the prototype are

  • Time saved: 14.5 hours/week per user
  • Cost: $0.42/day (85% cheaper than naive all-LLM approach)
  • ROI: 204x — payback in less than 1 day
  • Accuracy: 91% approve-without-edit rate (Day 7), trending to 95%+
  • Edit rate reduction: 40% → 12% after just 5 corrections
  • Builder proof: 15+ production AI skills in 2 months, zero coding background

What are the next steps for scaling your solution

  • Pilot (Month 1–2): 12 Claims staff, 2 champions per dept = 16 diverse twins
  • Expand (Month 3–6): All HK departments — UW, Finance, Renewal, Marketing, CS, Ops
  • APAC (Month 6–12): Playbook + template library for regional rollout
  • Global (Year 2): 55,000 staff × $31K = $170M annual value unlock
  • Champions-first strategy: diverse skills on Day 1 shows it works for ANY role

Layer 1: Factory (Brain Only)
• Claude-only — pure reasoning, no automation needed
• Runs once per user (10 min, ~$3 one-time)
• Re-run only if role/process changes
Layer 2: Daily Twin (Brain + Body)
• Brain (Claude): called ONLY when thinking needed
• Body (n8n): FREE orchestration, 400+ connectors
• Result: $0.42/day vs $2.80/day (85% cheaper)
Twin Profile: Self-Improving Memory
• Persistent .md config stores tone, thresholds, corrections
• All skills read it — behavior changes instantly
• Gets smarter every day (84% → 91% → 95%)
Why Level 5 Agentic (not just a chatbot)
• REASON: Analyzes role, identifies 4 daily tasks, spots SOP waste
• PLAN: Designs 3 architecture options, scores pros/cons
• DECIDE: Picks optimal design, assigns triggers per skill
• SELF-CORRECT: Quality gate fails? Loops back. Never proceeds without OK.
5 Winning Themes
1. Agent builds agents — 11 agents design YOUR unique twin
2. Cost-smart 2-layer — Claude brain + n8n body = 1/3 cost
3. Your twin ≠ my twin — same factory, different skills per role
4. Easy feedback, skills learn — approve/edit/reject, never repeats
5. Expandable, grows with you — promoted? Just ask for new skill
